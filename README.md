# StudyTask-Manager
A simple task manager for organizing study tasks with deadlines.

A simple Qt based task management application built in C++ using Qt6 and CMake.
It allows users to create, edit, delete, and manage study tasks through a clean and basic GUI.
Used on JSON storage and Qt recource System (.qrc)
